WUTUP API

Welcome

This is the WUTUP application API
This application allows you to listen to hip hop mixtapes
and manage your own playlist.
This API if for developers how wish to get and set their own
hip hop mixtapes and connect with Youtube API in order to get them
the main link for the WS is https://wutupws.herokuapp.com/
and the function will be the text after the '/'
Enjoy!

The Search Method

After the main link for the WS insert the following /search
the Method is GET
this function will bring you all the songs in the WUTUP library from Youtube API

Delete song

The Method is POST
in order to delete songs from a specific playlist
first you will need to add the following route /deleteSong
as parameters you will need to send the song id as a parameters that calls "songId"
if it works you will get an approve json that looks like that
{ "ok": 1,
"nModified": 0,
"n": 1,
"opTime": {
"ts": "6445556063621087235",
"t": 2 },
"electionId": "7fffffff0000000000000002"}
and the song will be delete from the DB.

Add Song

The Method is POST
in order to add song to the user's playlist you will need to add the route
/addSong
to add song to the DB of the user you will need to send a few parameters
"songId" - will be the song id - string
"songTitle" - will be the song title - string
"songAuthor" - will be the song author - string
"yearOfCreation" - will be the year of creation - number
"image" - will be a link to the image of the song - string
after you will send the following parameters you will get the same OK status
you got in the delete method.
